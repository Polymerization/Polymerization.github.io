How this website was built

I initially started using Hugo to build a simple blog styled website, but I didn't like how complex it was. I just wanted to generate something very simple. I was inspired by gingerBill, and made a static site generator, using the odin programming language. I'm happy with how the website turned out!

The (current) minimal styling is intentional and I think it is just me trying to rebel against how complex things are these days. I'll probably redecorate it in the future so I'll save a screenshot of what it looks like now so I can remember how good it looked!
